# Prompt dla Gemini — generowanie animowanych GIFów ćwiczeń

## Instrukcja dla Gemini

Wygeneruj animowane GIFy dla każdego ćwiczenia z listy poniżej.

**Wymagania techniczne:**
- Format: animowany GIF (looping)
- Rozmiar: 320×320 px
- Tło: jasne, neutralne (białe lub jasnoszare)
- Postać: realistyczna sylwetka człowieka, widok z boku lub ¾ — pokazująca ruch
- Styl: schematyczny ale czytelny (jak instrukcja fizjoterapeutyczna)
- Klatki: 6–12 klatek, płynna pętla
- Nazwa pliku: `{id}.gif` (podane przy każdym ćwiczeniu)

**Dla każdego ćwiczenia:**
> Wygeneruj animowany GIF o rozmiarze 320x320 px, pętla. Tło białe. Realistyczna sylwetka człowieka w stroju sportowym. Styl instruktażowy.

## Jak używać

1. Skopiuj prompt dla każdego ćwiczenia i wyślij do Gemini/Imagen/innego generatora
2. Zapisz wynikowy GIF pod nazwą `{id}.gif` (np. `101.gif`)
3. Umieść pliki w `/home/niemczyt/src/robopong-app/frontend/static/exercises/`
4. Frontend automatycznie spróbuje załadować `/static/exercises/{id}.gif`

## Alternatywne narzędzia

- **Gemini** (Google AI Studio) — image generation
- **Imagen 3** — Google's image model, handles animation
- **DALL-E 3** — pojedyncze klatki, składanie GIFa ręcznie
- **RunwayML / Pika Labs** — video → GIF conversion
- **Stable Diffusion + AnimateDiff** — lokalne generowanie animacji
